# Amazon Clone

A responsive e-commerce UI clone of Amazon's homepage built using HTML and CSS.

## Features
- Responsive navbar with search bar and cart icon
- Hero banner section
- Product grid layout using Flexbox
- Hover effects on cart icon and product cards
- Footer with multiple sections

## Technologies Used
- HTML
- CSS (Flexbox)
- Font Awesome icons

## What I Learned
- Building responsive layouts using Flexbox
- CSS hover effects and pseudo-classes
- Structuring a multi-section webpage
